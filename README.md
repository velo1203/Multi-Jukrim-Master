# 죽림고수 (Multi Jukrim Master)

사방에서 날아오는 표창을 피하는 실시간 멀티플레이어 웹 게임입니다.

## 게임 소개

여러 플레이어가 같은 맵에서 동시에 플레이하며, 상하좌우에서 쏟아지는 표창을 피해 가장 오래 생존하는 사람이 승리합니다. 시간이 지날수록 표창의 속도와 수가 증가합니다.

## 기능

- 실시간 멀티플레이어 (Socket.io 기반)
- 방 목록에서 방 선택 후 입장
- 최대 8명 동시 플레이
- 목숨 3개, 모두 소진 시 탈락
- 생존 시간 기반 점수 집계
- 모바일 터치 조작 지원

## 기술 스택

| 분류 | 기술 |
|------|------|
| 서버 | Node.js, Express |
| 실시간 통신 | Socket.io |
| 클라이언트 | HTML5 Canvas, Vanilla JS |
| 아이콘 | Font Awesome 6 |

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 서버 실행
node server.js
```

브라우저에서 `http://localhost:3000` 접속

같은 네트워크의 다른 기기에서는 `http://<서버 IP>:3000` 으로 접속합니다.

## 조작법

| 입력 | 동작 |
|------|------|
| `W` / `↑` | 위로 이동 |
| `S` / `↓` | 아래로 이동 |
| `A` / `←` | 왼쪽으로 이동 |
| `D` / `→` | 오른쪽으로 이동 |
| 터치 드래그 | 모바일 이동 |

## 게임 흐름

1. 닉네임 입력 후 **방 목록 보기** 클릭
2. 열린 방에 입장하거나 **새 방 만들기**
3. 방장이 **게임 시작** 버튼 클릭
4. 표창을 피하며 마지막까지 생존
5. 게임 종료 후 점수 확인 및 재시작
